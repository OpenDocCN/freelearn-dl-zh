

# 第十一章：使用 PowerPoint 第三方外加组件

到目前为止，你可能已经意识到 PowerPoint 已经成为一个非常强大的创意应用。它拥有符合大多数创作者需求的功能和工具，但如果你对 PowerPoint 的使用变得密集，你可能会发现一些工作流程可以改进，以使你的日子更加高效。这就是你可以考虑使用第三方外加组件的时候。如果你是大型组织的一部分，在安装任何外加组件之前，请确保与你的 IT 部门联系。一些 IT 设置会阻止第三方安装。

市面上有大量的 PowerPoint 外加组件——其中一些是免费的，而另一些则需要购买。当我反思我应该包括哪些外加组件在本章中时，我决定以信任和安全作为我的标准。我想确保开发者是值得信赖的，并且他们的外加组件对您的电脑是安全的。

因此，本章中建议的所有外加组件都是我认识的人——我甚至在他们现实生活中见过！他们没有付钱让我谈论他们的产品，但他们确实给了我一个许可证或与我分享了他们付费产品的某些视觉元素，而这些产品我之前并没有使用。我知道他们的产品在演示设计行业中非常受欢迎。我选择在本章中介绍的外加组件创作者如下：

+   BrightCarbon

+   Neuxpower

+   Billion Dollar Graphics

+   PPTools

# 技术要求

本章中介绍的外加组件已知与许多 Windows 和 macOS 桌面版本的 PowerPoint 兼容。我建议您访问开发者的网站以获取更多兼容性详情（见*进一步阅读*）。请注意，由于 PowerPoint 的订阅版本持续更新，因此本章中显示的截图可能与您的应用程序版本不同。

## 使用 BrightSlide（免费）

这个插件最初是为了帮助 BrightCarbon 设计师使他们的设计工作流程更加高效而创建的，添加了用户需要的快捷键和功能，但这些功能在 PowerPoint 中不可用。他们的第一个核心价值观是“*分享我们的知识和时间*”，因此他们很快决定将他们宝贵的插件与世界分享。不用说，我们的行业专家迅速将消息传播给所有演示文稿创建者和设计师。您可以通过提供的链接在他们的产品页面找到 BrightSlide。

BrightSlide 插件可能被选为我们行业中免费提供的最完整的工具。它包含九个工具类别，每个类别都包含许多功能。在撰写本书时，该插件已超过 60 个不同的功能，用于提高您的工作流程和生产力。当您尝试使用 BrightSlide 时，这可能会不同，因为 Jamie 和 BrightCarbon 产品团队会根据设计师或插件用户的需求定期添加功能。

要使用 BrightSlide，首先，您需要从产品页面（[`www.brightcarbon.com/products/`](https://www.brightcarbon.com/products/)）下载 Windows 或 macOS 安装程序，并在您的计算机上安装它。当您打开 PowerPoint 应用程序时，您将看到一个新的**BrightSlide**（**1**）选项卡（*图 11.1*）：

让我们探索您在新选项卡中会发现的主要功能类别：

+   **对齐**（**2**）：这个类别包含了一些工具，可以为您提供更多控制和选项来对齐您的幻灯片对象，而不仅仅是 PowerPoint 中标准对齐工具所能提供的。例如，您只需点击几下就可以创建一个形状网格。您甚至可以选择将对象对齐到第一个或最后一个选定的对象。

+   **格式**（**3**）：这个类别中的工具将帮助您一次性编辑多个对象或复制复杂的格式。这个类别可以帮助您将大小、位置、格式、调整和动画从一个对象复制到任何其他对象。

+   **选择与对象**（**4**）：这个类别旨在使您的生活更加便捷。根据特定的特征选择对象，例如填充或轮廓颜色，隐藏或显示幻灯片对象而无需滚动到您的**选择**面板，以及交换对象，这只是这个组中的一些亮点。您激动了吗？

+   **动画**（**5**）：这个类别中的工具在尝试重新创建复杂动画时可以为您节省大量时间。您可以使用 BrightSlide 的一些起始序列，或者将您自己的添加到库中。

+   **文件与母版**（**6**）：这个类别包含许多工具，可以帮助您设置模板，例如设置参考线、编辑您的主题颜色、设置自定义颜色、批量分配布局、比较多个演示文稿等等。这些功能可以处理大多数重复性任务。

+   **设置 & 帮助**（**7**）：在这里您可以找到功能区添加自定义选项和快捷键。您还可以访问两个功能子类别：

    +   **设置** – 被动功能：这些功能不会出现在功能区中，并且会修改默认的 PowerPoint 行为，您可以选择开启或关闭它们。例如，在复制幻灯片时可以复制注释，或者一次性调整多行的大小。

    +   **帮助** – 上下文工具：当您在幻灯片对象（如表格或线条）上右键单击时，这些工具将变得可用，允许您在演示文稿中移动部分内容，并轻松检查文本和填充颜色之间的对比度。

如果您想了解完整的详细信息，他们网站上有一个**帮助**部分，列出了功能及其描述，但我将分享一些我知道的工具，即使是兼职演示文稿设计师也应该使用这些工具来节省时间。为了帮助您找到它们，我将在以下子节中引用功能区上的组名。

### 对齐组

这个组有如此多的对齐功能，以至于几乎不可能在它们之间做出选择。我最终选择了三个我认为特别有趣的功能，以帮助您快速将多个对象叠加在一起，一个压一个，将它们分布到网格中，或者创建一个新的网格。

第一个功能帮助我们叠加所有选定的对象，并命名为**对齐对象中心与中间**（**1**）（*图 11.2*）：

![图 11.2 – 对齐组中的“对齐对象中心与中间”功能](img/B32063_11_02.png)

图 11.2 – 对齐组中的“对齐对象中心与中间”功能

在选择多个对象（以我们的例子中的六边形为例）后，只需在功能区中单击该功能，就可以一次性将所有对象叠加（**2**）。正如您在**选择**面板（**3**）中可以看到的，所有形状都出现在幻灯片上。您可以使用这种类型的对齐方式来制作动画，每次只显示一个形状，始终保持在同一位置。

现在，让我们在幻灯片上使用六个没有特定对齐方式的散布六边形，并使用这次的对齐组中的**分布到网格**工具（**1**）。BrightSlide 将对齐并分布它们（**2**），并打开**分布到网格**对话框（**3**），在其中您可以调整网格（*图 11.3*）：

![图 11.3 – 对齐组中的“分布到网格”功能](img/B32063_11_03.png)

图 11.3 – 对齐组中的“分布到网格”功能

您可以调整**行数**和**列数**（**4**），设置**间距**单位（**5**）为英寸、厘米或像素，并调整幻灯片对象之间的**水平**和**垂直**间距（**6**）。**预览**复选框（**7**）默认选中，在点击**确定**按钮（**8**）确认选择之前查看您的网格外观非常有帮助。在撰写此章节时，此功能只能用于相同大小的对象。

尽管 PowerPoint 的**智能参考线**工具已经帮助我在网格上对齐和分布形状很长时间了，但这个 BrightSlide 功能每天可以节省很多点击，使其更加高效。

我想要向您展示的**对齐**组的第三个功能要求您从幻灯片上的单个对象或一组对象开始（**1**）并使用**分割与对齐**功能（**2**）（*图 11.4*）：

![图 11.4 – 对齐组中的分割与对齐功能](img/B32063_11_04.png)

图 11.4 – 对齐组中的分割与对齐功能

此功能创建了一个由大小均匀且间隔均匀的组或选定对象的副本组成的网格。在我的例子中，它创建了一个大小均匀且间隔均匀的矩形网格（**3**），同时打开**分割与对齐**对话框（**4**）。您可以根据需要调整**列**（**5**）和**行**（**6**）的设置。您看到的**间距**（对象之间的间隔）、**宽度**和**高度**（**7**）的数字受您为**单位**选择的设置的影响。在测试此功能时，我发现使用英寸或厘米不如使用像素灵活，像素是更小的单位。我最好的建议是在第一次尝试时进行几次测试。这将是一次明智的投资，因为 BrightSlide 会记住您之前的设置，并在您下次使用**分割与对齐**功能时使用它们。

**对齐**组中提供了许多更多功能。要了解哪些功能对您的演示文稿工作类型有用，最好的方法是尝试使用它们，或者至少查看 BrightSlide 网站上的**帮助**部分。

### **格式**组

此类别也有一个很棒的特性列表可供使用，但由于空间有限，我只想指出**匹配大小**功能（**1**），它允许您通过单次点击快速将幻灯片上选定的对象（或两个中的任意一个）的**宽度**和**高度**设置为相同（**2**）（*图 11.5*）：

![图 11.5 – 格式组中的匹配大小功能](img/B32063_11_05.png)

图 11.5 – 格式组中的匹配大小功能

为了确保您获得期望的结果，当您将鼠标悬停在功能按钮上时（**3**），请注意提示信息。正如我们所见，仅选择一个对象将参考幻灯片的尺寸——想象一下您能多快地裁剪并制作一个全幻灯片图像！

默认情况下，最后选定的对象成为之后创建的任何对象的尺寸参考。在我的例子中，如果我先选择图像（**4**）然后选择**事件 3**按钮（**5**），**匹配大小**功能将使图像与动作按钮大小相同。请注意，使用此功能调整大小的图像将被扭曲，而不是裁剪以适应大小，并且将保持相同的**宽高比**。

### **选择**与**对象**和**动画**组

这两个类别充满了所有设计师都可以利用的生产力工具。为了节省空间，我将只对几个功能给出一个高级定义，以便您了解它们可能有多么强大（*图 11.6*）：

![图 11.6 – 选择对象和动画组的高亮](img/B32063_11_06.png)

图 11.6 – 选择对象和动画组的高亮

+   **选择对象** ( **1** )：此功能允许根据某种格式样式选择幻灯片上的所有对象，这样您可以轻松地一次性更改它们的格式。

+   **拆分文本** ( **2** )：此功能允许您通过段落或句子将文本内容拆分为多个对象。

+   **文本边距** ( **3** )：此功能允许您一键删除所有文本边距。无论您只想修复一个对象，还是所有幻灯片上的对象，或者是母版上的对象，都能满足您的需求。

+   **我的动画** ( **4** )：此功能允许您保存并重复使用自己的动画库。在为对象创建复杂的动画方案后，您可以将其从所选对象保存下来以供将来使用。

+   **实用工具** ( **5** )：在这个列表中，您将找到创建瀑布延迟、更有效地管理运动路径以及删除所选幻灯片或演示文稿中所有动画的功能（**6**）。当您意识到动画是从各种来源被引入您的演示文稿时，最后一个功能可以节省大量时间。

了解这些功能的最佳方式是通过在您的演示文稿文件中尝试它们。花时间阅读在功能上悬停时出现的工具提示也会对您有很大帮助。

### 文件和母版组

尽管文件和母版组可能不像其他组那样有趣，但我必须说，这是我找到的一些我最珍视的功能的地方，这些功能适合我从事的工作类型。

我将突出显示对我非常有用的那些（*图 11.7*）：

![图 11.7 – 文件和母版组的高亮](img/B32063_11_07.png)

图 11.7 – 文件和母版组的高亮

+   **参考线** ( **1** )：当我收到带有太多参考线的文件，或者当我创建临时参考线时，**删除参考线**功能可以节省大量宝贵的时间。我们可以用它来选择我们想要删除的参考线（母版、布局或正常视图），或者删除所有参考线。

+   **主题颜色** ( **2** )：在这个列表中，我最常用的两个功能是：

    +   **颜色对比报告** ( **3** ) 创建一个新的幻灯片，允许我根据形状填充评估文本对比度可访问性。

    +   **删除最近颜色**功能（**4**）从最近列表中删除所有颜色样本。在测试一些非品牌颜色或清理模板文件以发送给客户之前，这非常有价值。

+   **批量**（**5**）：此列表中的每个功能都很宝贵。我特别喜欢的是当你必须处理复杂演示文稿时，布局被误用的情况，即**分配布局**。从 *一个* 窗口中，你将能够更改文件中的任何幻灯片的母版或布局。然而，这意味着你必须熟悉母版和布局使用的名称。你仍然可以在做出任何更改之前通过幻灯片导航来查看它们的内容。

+   **审查**（**6**）：此列表中的两个功能特别有价值：

    +   而不是试图绝望地使用 PowerPoint 的原生工具更改文件的语言，有时效果可疑，我使用 **演示文稿语言**。只需 *一个* 点击，它就会更改 *整个* 演示文稿的语言。

    +   至于 **演讲者备注**，它允许你为所有演讲者备注获得可靠的字数统计，并且可以一键删除所有备注。

+   **导出**（**7**）：此列表中的功能解决了 PowerPoint 的一些重要缺失功能。从所选幻灯片中创建新的演示文稿文件可以节省大量时间。同样，将所选幻灯片导出为图像或导出演示文稿中的所有媒体文件也是如此。

BrightSlide 插件是任何使用 PowerPoint 创建视觉效果的宝藏。我最好的建议是下载此插件并尝试一下。你没有任何损失（它是免费的！），但你可以在效率和生产力方面获得很多收益。如果你想知道为什么 BrightCarbon 将此免费提供给所有用户，那是因为他们的公司文化，他们相信与社区分享。

## 使用 BrandIn（免费增值和付费）

许多组织在保持品牌一致性方面感到困难。大多数情况下，他们已经投资于创建一个使用公司颜色的品牌 PowerPoint 模板，在示例幻灯片中提供图形元素，并使用一组标准布局进行规划——但用户往往没有足够的 PowerPoint 技能来正确地导航整个模板，或者没有时间阅读数十页的品牌指南。

正因如此，才创建了 BrandIn。它帮助组织提供对资产库的便捷访问，同时帮助用户保持品牌一致性，所有这些都在 PowerPoint 内完成。而且这里有一个巨大的好处：你可以在 PowerPoint 和 Word 中使用它，在 Windows、macOS 和在线上使用。

此插件解决了组织中许多最令人烦恼的演示文稿创建问题，例如以下问题：

+   用户浪费了数小时试图制作符合品牌规范的幻灯片，或者甚至只是寻找相关内容

+   用户添加了非品牌和/或受版权保护的图片，而不是来自品牌库的图片

+   存在着从其他演示文稿复制幻灯片、臃肿的模板和母版的问题

BrandIn 可在 Microsoft AppSource 商店找到，并且也可以从 **M365 管理中心** 向组织用户提供。许可模式是按年订阅，根据你拥有的用户数量付费，类似于 M365 许可证。提供不同级别的许可证，具有不同的功能集，包括一个为一到四人团队提供的免费计划。有关更多信息，请参阅 *进一步阅读* 部分，其中包含其网站的链接，并联系他们讨论您的许可需求。

如前所述，如果你无法访问 PowerPoint 或 Word 的 **首页** 选项卡中的 **添加组件** 商店，你可能需要从你的 IT 管理员那里获得支持以在你的组织中部署添加组件。部署后，有一些初始设置任务需要完成，以便 BrandIn 的魔法发挥作用：

1.  创建一个专门用于所有创意企业资产的新的 SharePoint 网站，并决定谁将负责内容维护。你也可以使用现有的网站或 **组织资产库**（**OAL**）。

1.  精选你的实际内容，并将其分类到以下类别中：`模板`、`幻灯片`、`图片`、`图标` 和 `图形`。

1.  在您的新 SharePoint 网站中为每个类别创建一个文档库，并将您整理好的内容添加到每个库中，按命名文件夹组织。

1.  在 BrandIn 中创建一个新的工作空间，并链接文档库。

如你或许已经意识到，你首先需要投入一些时间，这样你的团队能够节省大量的创作和设计时间，并保持品牌一致性，从而获得投资的最佳回报。

我可以说，最耗时的不是创建 SharePoint 网站和文档库，尤其是如果你并不真正关心拥有一个品牌网站的话——仅仅因为我已经很久没有这样做，这花了大约 15 分钟。最耗时的部分是收集你已有的内容，整理它们，以确保你只提供与你的品牌一致的内容，然后在 BrandIn 中连接库。当然，确保定期维护资产，以保持它们与品牌的一致性。我的建议是为这项任务指定你团队中的某个人。

当您想测试文本在布局上的外观时，您可以使用 Office 应用程序中的一个隐藏功能来创建填充文本。要创建看起来像拉丁文的文本，您需要输入 `=lorem(x,y)` ，其中 `x` 是段落/项目符号的数量，`y` 是每行的句子数量，然后按 *Enter* 键。

如果您想生成可读的文本，可以使用 `=rand(x,y)` ，这将生成句子 **The quick brown fox jumps over the lazy dog**，并重复应用于所有段落/项目符号和句子。为什么是这个句子？因为它使用了字母表中的所有字母，让您可以看到您的字体如何显示。请注意，在 PowerPoint 中，即使您使用的是另一种用户界面语言，随机文本也可能相同——这就是我在法语中的情况。但在法语 Word 中，生成的随机文本讨论的是 Word 功能。

# PPTools

为什么在讨论设计插件之后，我会回到生产力工具上来？简单来说，因为 **PPTools** 背后的这位人物在我们行业中备受尊敬。Steve Rindsberg 自 1997 年以来一直在创建 PowerPoint 插件和帮助文章！2005 年在一场会议中第一次见到他时，我就交了一个好朋友。

他的网站（ [`www.pptools.com/products.htm`](http://www.pptools.com/products.htm) ）上有许多年来为满足用户特定需求而创建的插件，这些需求是 PowerPoint 本身功能无法满足的。您还可以在他的网站上找到一篇关于插件及其类型的文章 – 我已在 *进一步阅读* 中添加了链接。而不是试图确定哪些要包含在本章中，我只是请 Steve 告诉我他的两个最受欢迎的插件，一个是免费的，一个是付费的。我们将在下一节中首先介绍免费的插件。

## 使用 THOR（免费）

**THOR** 插件有一个简单的焦点：记住幻灯片上对象的大小和位置，这样您就可以将其应用 – 或者像 Steve 会说的那样“锤击” – 到您的演示文稿中其他一个或多个选定的对象上。至于名称，是的，他指的是北欧神话中挥舞锤子的神，这反映了他极大的幽默感和他的文化素养。

要安装此插件，只需导航到网站上的 **免费 PPTools 插件** 部分（见 *进一步阅读* ）并从列表中选择 **THOR - 锤子**。然后，按照 **下载和安装 THOR** 部分的说明操作。安装完成后，您将在功能区看到一个新的 **PPTools** 选项卡（ **1** ）（见 *图 11.25* ）:

![图 11.25 – 带有锤子工具的 PPTools 选项卡](img/B32063_11_25.png)

图 11.25 – 带有锤子工具的 PPTools 选项卡

**Thor** 组有一组简单的按钮。以下是每个按钮的简要说明：

+   **记忆**（ **2** ）：这是您想要记住幻灯片对象的大小和位置时需要点击的第一个按钮

+   **Hammer**（**3**）：这是在选择了想要应用相同大小和位置的另一个或多个对象后要点击的按钮。

+   **帮助**（**4**）：此按钮打开一个 `.pdf` 文件，提供您所需的所有关于插件的信息，而无需返回网站。

+   **关于**（**5**）：这提供了访问常规许可信息、支持链接和诊断功能。

+   **选项**（**6**）：此按钮允许您通过更改**THOR 选项**对话框中的选项来配置 THOR 的工作方式。

    +   **对齐**（**7**）：点击单选按钮选择要使用的参考点。在这个例子中，THOR 将记住对象的左上角位置。

    +   **调整大小**（**8**）：此选项允许您记住对象的位置（如果复选框未选中）或记住如何调整对象的大小，以及如何将它们移动到相同的位置。

    +   **不变形**（**9**）：当您决定调整幻灯片对象的大小以避免变形时，此选项非常重要。

    +   **静音模式**（**10**）：此选项仅避免弹出对话框，显示已记住的对象的确切位置和大小。如果您想了解更多信息，请参阅**帮助**文档。

    +   当然，还有一个**确定**按钮来确认您的设置。

## 使用 PPTMerge（付费）

**PPTMerge** 插件是一个非常有用的工具，可以帮助您从 Excel 文件中合并数据，创建定制的单个幻灯片或完整的演示文稿。如果您想知道这如何有帮助，这里有一些例子：

+   创建包含每张幻灯片上特定人员名称的演示文稿文件。

+   使用证书 PowerPoint 模板。然后你可以合并要在每个证书上显示的名称列表。

+   通过将一些关键元素合并到 PowerPoint 中创建的视觉模板中，创建你的社交媒体帖子。这个令人惊叹的创意归功于来自 My TechGenie 的 Catharine Richardson（见*进一步阅读*）。

这个节省时间的插件在 PPTools 网站的产品列表中作为一次性购买提供。Steve 的 PPTMerge 页面上的一个客户评价说他们每年只使用一次，但仍物有所值。

安装后，你可以在功能区中的**PPTools**标签页找到 PPTMerge（**1**）（*图 11.28*）：

+   在你的 Excel 文件中创建，单元格**A1**有相同的`:Name:`标题，这样插件就会知道在数据文件中查找什么。在下面的行中添加所有需要的名称。
