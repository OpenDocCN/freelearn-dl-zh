# 前言

**大型语言模型**（**LLMs**），如 GPT-4、Bard 和 Claude 的出现，标志着人工智能（**AI**）能力的巨大转变。这些模型由庞大的数据集和计算能力驱动，可以生成令人惊叹的类似人类的文本并参与复杂的对话。然而，它们的潜力只有通过提示工程这一艺术——精心设计激活模型的提示过程——才能完全实现。提示编码了指令、上下文、示例和限制条件，以引导模型的能力执行特定任务。掌握提示工程解锁了 LLM 在众多应用中的巨大潜力，从内容创作到数据分析等。

本书作为不同领域提示工程的实用指南。通过具体的例子和现实世界的案例研究，读者将学习如何将问题分解为离散的提示、提供相关背景知识、迭代优化提示以及塑造模型输出。这一过程使读者掌握利用经过适当工程化的提示挖掘 LLM 多面性的策略。遵循本书中的技术，任何人都可以利用 GPT-4 和 Claude 等 AI 系统的解决问题的技能，构建创造真实价值的解决方案。

# 本书涵盖内容

*第一章*，*理解提示与提示技术*

这本入门章节全面概述了 LLM 提示和提示工程的基础。它探讨了提示的组成部分、不同的提示技术、LLM 参数以及构建有效提示的系统框架。本章还讨论了需要解决的问题，如冗长和不一致性。通过为您提供关于提示工程和如何引导 LLM 行为的核心知识，本章为在后续章节中利用 AI 的强大功能奠定了基础。

*第二章*, *利用 AI 辅助更快地创作引人入胜的内容*

本章探讨了利用 ChatGPT 等 AI 工具生成、概述和草拟包括社交媒体帖子、销售文案、视频脚本和文章在内的内容初稿。它涵盖了提供上下文和示例以引导 AI，个性化信息，定制语气和声音，以及精炼原始 AI 输出的内容。尽管 AI 通过这些技术显示出增强人类创造力和内容创作生产力的潜力，但人类监督仍然至关重要。关键课程集中在深思熟虑地将 AI 辅助与人类创造力和意图相结合，以开发引人入胜且高质量的内容。

*第三章*, *使用 ChatGPT 和其他实用示例创建和推广播客*

本章提供了利用 AI 工具（例如 ChatGPT）进行实际任务的实用示例，例如制作引人入胜的播客和面试问题。它探讨了识别播客主题、潜在嘉宾和推广内容的提示和技术。对于面试，它涵盖了面试官和应聘者如何使用 AI 来策略性地提出相关问题和深思熟虑的答案。关键课程集中在使用 AI 加速播客和面试的准备、想法生成和内容创作，同时增强人类创造力。

*第四章*, *用于创意写作的 LLMs*

本章探讨了作家如何利用 ChatGPT 等 AI 工具来提升创意写作过程的各个方面。它提供了如何构建提示以生成小说中的想法、角色和情节，以及创作原创诗歌的技巧的例子。关键课程集中在利用 AI 激发想象力，同时保留作者的视野和声音。在人类创造力和 AI 辅助的正确平衡下，这些模型可以加速想法生成，通过编辑改进草稿，并开辟新的创意领域。

*第五章*, *从非结构化文本中解锁洞察力 - 文本分析的 AI 技术*

本章探讨了 AI 技术（如情感分析、数据分类、数据清洗和模式匹配）在从非结构化文本中提取洞察力方面的关键应用。它提供了使用这些技术执行诸如衡量内容中的情感、分类数据、解决不一致性和提取结构化信息等任务的例子。关键课程集中在利用 AI 自动化定性数据分析，节省时间和精力，同时提高准确性。有了正确的技术，AI 使任何人都能从非结构化文本数据的激增中解锁价值。

*第六章*, *LLMs 在教育* *和法律中的应用*

本章展示了 ChatGPT 等 AI 系统在教育和法律领域的应用。它提供了使用这些工具生成个性化课程材料、练习题和针对学习目标的评分标准的示例。对于法律专业人士，本章探讨了利用 LLM 进行研究、起草文件、知识产权管理、培训法律学生以及其他新兴用例。然而，对 AI 响应的人类验证仍然至关重要。当深思熟虑地实施时，ChatGPT 等工具显示出巨大的潜力，通过自动化重复性任务和提高生产力来协助教育、法律和其他领域的专业人士。

*第七章*，*AI 配对程序员崛起 - 与智能助手合作以实现* *更好的代码*

类似于 GPT-4 这样的 LLM 正在通过生成功能代码块、解释代码、调试、优化性能以及在不同编程语言之间进行翻译来改变编码方式。本章提供了案例研究，展示了如何使用 AI 快速开发网站代码和 Chrome 扩展，使开发者能够专注于设计而非机械的编码任务。GitHub Copilot 等 AI 编码助手利用 GPT-3 和 GPT-4 提供针对开发者需求的自主代码生成。虽然 AI 可以加速开发，但在部署前仍需要人工审查和精炼生成的代码。AI 不太可能很快完全取代开发者，但它可以增强人类在编码中的创造力和解决问题的能力。未来将涉及流畅的人机协作，程序员和助手将协同工作。

*第八章*，*对话式 AI* *–* *构建智能* *聊天机器人 LLM*

由 GPT-3/4 和 Claude 等 LLM 驱动的聊天机器人正在改变对话式 AI，并使数字体验更加自然、更接近人类。正如本章详细示例所示，这些强大的生成模型使机器人能够真正理解自然语言，与用户进行自由流动的对话，并完成从商业交易到个性化评估的复杂工作流程。

解锁其能力的关键是深思熟虑的提示工程。开发者可以将关键上下文、领域知识、业务逻辑、数据源等内容注入提示中，以塑造机器人的行为。虽然在与游乐场互动中可以一瞥其潜力，但基于 LLM API 构建的定制解决方案则打开了更多可能性。

*第九章*，*构建更智能的系统* *–* *高级* *LLM 集成*

本章探讨了将 LLM 集成到实际工作流程中的各种技术，以解锁新的可能性。易于使用的模板，如 SheetSmart，简化了在电子表格中设置公式以批量提示 LLM（如 GPT-3.5）。更强大的自动化平台，如 Zapier 和 Make，使连接 Web 应用程序到 LLM API 的管道成为可能。这允许通过将数据源摄入 LLM 来自动化生成竞争情报简报等流程。

为了实现完全定制，开发者工具如 LangChain、Flowise 和 Langflow 提供了构建涉及推理、对话和上下文推荐的复杂 LLM 应用的框架。本章的示例演示了从客户数据中提取见解以丰富 CRM 系统以及使用 LLM 与 PDF 文档进行对话的示例集成。

*第十章*，*生成式 AI* *–* *伦理与创新的交汇点问题*

生成式 AI 在信任、问责制、偏见风险、从生产力提升到就业转移的经济影响、威胁可持续性的巨大计算需求、微妙的社会风险以及围绕机器创造力的哲学问题等方面带来了深刻的挑战。解决方案需要通过设计伦理、算法评估、深思熟虑的法规、包容性治理和捍卫人权进行合作。今天在 AI 伦理和治理方面做出的选择将产生深远的影响。在人文主义的指导下，这些技术可以被引导用于提升和丰富社会。

*第十一章*，*结论*

提示工程代表了指导生成式 AI 系统（如 LLM）自动化任务并增强跨行业人类能力的突破。根据输出仔细调整提示是引导这些模型的关键，就像烹饪时调整配料一样。目前，人类专业知识仍然是监督 AI 初步能力所必需的。展望未来，条件提示和因果提示等技术可能会使更可靠、个性化的应用成为可能。医疗保健和其他领域，如工程和金融，展示了提示工程在协助专业人士方面的巨大潜力。然而，我们必须承认当前的局限性，并实施深思熟虑的治理来负责任地管理风险。整合模型与服务的平台以及通过在独特数据上训练来定制输出的工具将进一步扩大可能性。虽然专注于文本生成，但随着多模态 LLM 的发展，提示工程将变得更加多才多艺。

总体而言，本书提供了对技术方法的入门性概述，这些方法可以通过深思熟虑的提示工程激发几乎每个生活和工作方面的激动人心的可能性。
