# RAG 的组件

在*第二部分*，你将学习关于 RAG 系统关键组件以及如何使用 LangChain 来实现它们。你将探索使用 Gradio 与 RAG 交互以创建交互式用户界面，向量及其存储在增强 RAG 性能中的关键作用，以及评估 RAG 的定量方法和可视化技术。此外，你还将深入了解使用 LangChain 组件，如文档加载器、文本分割器和输出解析器，以进一步优化你的 RAG 管道。

+   *第六章*，*与 RAG 和 Gradio 的交互*

+   *第七章*，*向量及其存储在 RAG 中的关键作用*

+   *第八章*，*使用向量进行相似性搜索*

+   *第九章*，*定量和可视化评估 RAG*

+   *第十章*，*LangChain 中的关键 RAG 组件*

+   *第十一章*，*使用 LangChain 从 RAG 中获得更多内容*
