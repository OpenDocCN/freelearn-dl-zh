# 第一部分

# 规划你的演示内容与设计

在本书的第一部分，你将建立起对如何规划你的演示内容与设计的坚实基础。我们将讨论一些问题，这些问题将帮助你在你打开 PowerPoint 应用程序之前规划你的内容，以及行业最佳实践来设计更好的视觉元素。我们还将学习如何利用 PowerPoint 中的各种母版。到这一部分结束时，你将拥有开始规划你的内容并创建或调整你的演示模板所需的知识。

本部分包括以下章节：

+   *第一章* ，*分析你的受众和演示交付需求*

+   *第二章* ，*运用行业最佳实践设计更好的视觉元素*

+   *第三章* ，*利用 PowerPoint 幻灯片母版进行设计*

+   *第四章* ，*使用 PowerPoint 文档母版制作讲义和备注*
