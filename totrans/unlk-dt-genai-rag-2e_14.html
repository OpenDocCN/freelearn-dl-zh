Part 3

Implementing Agentic RAG

In Parts 1 and 2, you built a strong foundation in RAG: retrieval fundamentals, chunking strategies, embedding models, pipeline design, vector stores, similarity searching, evaluation techniques, and core LangChain components. This foundation is essential because RAG remains the core architecture underlying the most advanced AI applications being built today, serving as the backbone that agents, graphs, caches, and memory systems all extend and enhance.

With the foundation now securely in place, Part 3 prepares you for the cutting edge of agentic AI development. You’ll begin by integrating AI agents with LangGraph for more powerful control flows, then explore knowledge engineering through ontologies and graph-based RAG architectures that enable complex reasoning over structured data. The chapters progress into semantic caches for dramatically reducing latency and inference costs, followed by a deep dive into agentic memory systems. These represent the most advanced expression of RAG you can implement, transforming stateless agents into intelligent systems capable of learning and adapting over time. Through hands-on code labs, you’ll implement the CoALA memory framework, build procedural memory with LangMem, and integrate complete memory architectures into your RAG pipelines.

This part contains the following chapters:

  • Chapter 12, Combining RAG with the Power of AI Agents and LangGraph
  • Chapter 13, Ontology-Based Knowledge Engineering for Graphs
  • Chapter 14, Graph-Based RAG
  • Chapter 15, Semantic Caches
  • Chapter 16, Agentic Memory: Extending RAG with Stateful Intelligence
  • Chapter 17, RAG-Based Agentic Memory in Code
  • Chapter 18, Procedural Memory for RAG with LangMem
  • Chapter 19, Advanced RAG with Complete Memory Integration
