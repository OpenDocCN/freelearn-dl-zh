Using PowerPoint Third-Party Add-Ins

By now, you probably realize that PowerPoint has become a very powerful creative application. It has functions and tools that will fit most creators’ needs, but if your use of PowerPoint becomes intensive, you might find that some workflows could be improved to make your days more efficient. This is when you can consider using third-party add-ins. If you are part of a larger organization, make sure to check with your IT department before installing any add-ins. Some IT settings will block third-party installations.

There are a lot of PowerPoint add-ins available on the market – some of them are free, and others need to be purchased. When I was reflecting on which add-ins I should include in this chapter, I decided to use trust and security as my criteria. I wanted to make sure the developers were trustworthy, and their add-ins were safe for your computers.

Therefore, all the add-ins suggested in this chapter are from people I know personally – I have even met them in real life! They did not pay me to talk about their products, but they did give me a license or shared some visuals with me for paid products that I did not already use. I know that their products are hugely popular in the presentation design industry. The add-in creators I have chosen to feature in this chapter are as follows:

  • BrightCarbon
  • Neuxpower
  • Billion Dollar Graphics
  • PPTools

Technical requirements

The add-ins featured in this chapter are known to be compatible with many desktop versions of PowerPoint on Windows and macOS. I suggest that you visit the developers’ websites for more details on compatibility (see Further reading ). Be aware that since the subscription version of PowerPoint is updated on an ongoing basis, it is possible that the screenshots shown in this chapter will differ from your version of the application.

Using BrightSlide (for free)

This add-in was first created to help BrightCarbon designers make their design workflow more efficient, adding shortcuts and functionality that users neede d but that were not available in PowerPoint. With their first core value being “ Share our knowledge and our time ,” it was not long before they decided to share their valuable add-in with the rest of the world. Needless to say, our industry experts were quick to spread the word to all presentation creators and designers. You can find BrightSlide on their products page via the link supplied in Further reading .

The BrightSlide add-in would probably be voted the most complete tool available for free in our industry. It has nine tool categories, each containing many features. At the time of writing this book, the add-in has over 60 different features for improving your workflow and productivity. This might be different by the time you try BrightSlide because Jami e and the BrightCarbon product team ad d features regularly as their designers or add-in users voice their needs.

To use BrightSlide, first, you need to download the Windows or macOS installer from the product page ( https://www.brightcarbon.com/products/ ) and install it on your computer. When you open your PowerPoint application, you will see a new BrightSlide ( 1 ) tab ( Figure 11.1 ):

Figure 11.1 – The BrightSlide add-in tab in PowerPoint

Let’s explore t he main categories of features you will find in your new tab:

  • Align ( 2 ): This category has tools that give you more control and options to align your slide objects beyond the standard alignment tools found in PowerPoint. For example, you can create a shape grid with only a few clicks. You can even choose to align objects to the first or last selected object.

  • Animation ( 5 ): The tools in this category will save you a lot of time when trying to recreate complex animations. You can use some of BrightSlide’s starting sequences or add your own to the library.

The Align group

This group has so many alignment features that it’s almost impossible to choose between them. I finally set my choice on three that I find partic ularly interesting to help you quickly overlay many objects, one on top of the other, distribute them on a grid, or create a new grid.

The first feature helps us overlay all selected objects on top of each other and is named Align Objects Center and Middle ( 1 ) ( Figure 1 1.2 ):

Figure 11.2 – The Align Objects Center and Middle feature in the Align group

With several objects selected (hexagons in our example), just click on the feature in the ribbon to overlay all of them with one click ( 2 ). As you can see in the Selection pane ( 3 ), all the shapes are present on the slide. You can use this type of alignment for animations that reveal one shape after the other, always in the same position.

Let’s now use six hexagons spread out with no sp ecific alignment on the slide and use the Distribute to Grid tool ( 1 ) this time. BrightSlide aligns and distributes them ( 2 ) and opens the Distribute to Grid dialog box ( 3 ), in which you can adapt the grid ( Figure 1 1.3 ):

Figure 11.3 – The Distribute to Grid feature in the Align group

You can then adjust the number of Rows and Columns ( 4 ), set Spacing units ( 5 ) as inches, centimeters, or pixels, and adapt the Horizontal and Vertical spacing ( 6 ) between your slide objects. The Preview checkbox ( 7 ) is checked by default and is very helpful for viewing what your grid will look like before clicking on the OK button ( 8 ) to confirm your choices. At the time of writing this chapter, only same-sized objects can be used with this feature.

Even though PowerPoint’s Smart Guides tool has helped me align and distribute shapes on a grid for a long time, this BrightSlide feature saves many clicks in a day, making it much more efficient.

The third feature I want to show you from the Align group requires you to start from a single object or a group of objects on your slide ( 1 ) and use the Split & Align feature ( 2 ) ( Figure 11.4 ):

Figure 11.4 – The Split & Align feature in the Align group

This feature creates a grid of evenly sized and spaced copies of a group or the selected object. In my example, it created a grid of evenl y sized and spaced rectangles ( 3 ) while opening the Split & Align dialog box ( 4 ). You can adjust the settings as needed for Columns ( 5 ) and Rows ( 6 ). The numbers you see for Gutter (spacing between the objects) and Width and Height ( 7 ) are impacted by the setting you choose for Units . When testing the feature, I realized that using inches or centimeters was not as flexible as using pixels, which are much smaller units. My best advice is to do a few tests the first time you try it. It will be time well invested because BrightSlide will remember your previous settings and use them the next time you use the Split & Align feature.

Many more features are available in the Align group. The best way to know which ones would be useful for your type of presentation work is to try them out or at least review BrightSlide’s Help section on the website.

The Format group

This category also has a great list of features to work with, but since I have limited space, I’ll just point out the Match Size feature ( 1 ), which a llows you to quickly make both Width and Height the same for objects selected on a slide (or either one of the two) with a single click ( 2 ) ( Figur e 11.5 ):

Figure 11.5 – The Match Size feature in the Format group

To make sure you get the desired result, pay attention to the tooltip when you hover over the feature button ( 3 ). As we can see, selecting only one object will reference the size of the slide – imagine how quickly you can crop and make an image full slide!

By default, the last object selected becomes the size reference for any objects created afterward. In my example, if I first select the image ( 4 ) and then the Event 3 button ( 5 ), the Match Size feature will make the image the same size as the action button. The Selection & Object and Animation groups

Those two categories are filled with productivity tools that all designers can leverage. For the sake of space, I’ll only give a high-level definition for a few features to give you an idea of how powerful they can be ( Figure 11.6 ):

Figure 11.6 – Highlights of the Selection & Object and Animation groups

  • Select Objects ( 1 ): This feature allows selecting all objects on a slide according to a certain formatting style so you can easily change their formatting all at once.
  • Split Text ( 2 ): This feature allows you to split the text content into multiple objects either by paragraph or by sentence.
  • Text Margins ( 3 ): This feature allows you to remove all text margins in one click. Whether you want to fix only one object, or objects on all the slides, or the ones on the masters, you’re covered.
  • My Animations ( 4 ): This feature allows you to save and reuse your own animation library. After creating a complex animation scheme for an object, you can save it from your selected object for future use.
  • Utilities ( 5 ): In this list, you will find features to create waterfall delays, manage motion paths more efficiently, and delete all animations on selected slides or in the presentation ( 6 ). The best way to get to know the features is by trying them in your presentation files. Taking time to read the tooltips available when hovering over the features will also help you a lot.

The File & Master group

Even though the File & Master group might not sound as interesting as others, I must say it is where I have found some of the features I treasure the most for the type of work I do.

I’ll highlight the ones that have been very useful for me ( Figure 11.7 ):

  • Guides ( 1 ): When I receive a file with too many guides, or when I create temporary guides, the Delete Guides feature saves a lot of precious time. We can use it to select which guides we want to delete (master, layout, or normal view) or delete them all.
  • Theme Colors ( 2 ): There are two features I use the most in this list:

  • Export ( 7 ): The features in this list address some of PowerPoint’s important missing features. The ability to create a new presentation file from selected slides is a huge time-saver. Same for the ability to export selected slides as images or export all the media files in the presentation.

Using BrandIn (Freemium & paid)

Many organizations struggle with staying on brand. Most of the time, they have invested in creating a branded PowerPoint template that uses their corporate colors, provides graphic elements in sample slides, and was plann ed with a set of standard layouts – but users often don’t have the PowerPoint proficiency required to navigate the whole template properly, or the time to read dozens of pages of a brand guide.

That is why BrandIn was created. It helps organizations offer easy access to a library of assets while helping users stay on brand, all within PowerPoint. And here is a great perk: you can make it work within PowerPoint and Word, on Windows, macOS, and online.

The add-in fixes many of the most annoying presentation creation problems in organizations, such as the following:

  • Users waste hours struggling to make slides brand-compliant, or even just finding relevant content
  • Users add off-brand and/or copyrighted images instead of those from the brand library
  • There are problems related to slides copied from other presentations, bloated templates , and masters

BrandIn can be found in the Microsoft AppSource store and can also be made available to an organization’s users from the M365 admin center . The licensing model is a yearly subscription paid for the number of users you have, much like the M365 licenses. Different levels of licenses with varying feature sets are available, including a free plan for teams of one to four people. For more information, please see the Further reading section for a link to their site and contact them to discuss your licensing needs.

As mentioned before, you may need support from your IT admins to deploy the add-in in your organization if you don’t have access to the Add-ins store in the Home tab of PowerPoint or Word. Once deployed, there are some initial setup tasks to accomplish to make BrandIn’s magic happen:

  1. Create a new SharePoint site dedicated to all your creative corporate assets and decide who will be responsible for content maintenance. You can also use an existing site or an organization asset library ( OAL ).
  2. Curate your actual content and sort it into the following categories: Templates , Slides , Images , Icons , and Graphics .
  3. Create a document library for each category in your new SharePoint site and add your sorted content to each one, organized by named folders.
  4. Create a new workspace in BrandIn and link the document libraries.

As you probably realize, you need to first invest some time, so your team saves a lot of creation and design time and stays on brand, so you get the best return on your investment.

I can say that it’s not the SharePoint site and document libraries’ creation that will take the most time, especially if you don’t really care about having a branded site – it took me about 15 minutes just because I had not done it in a long time. The most time-consuming part is gathering what you already have, cleaning it up so you are only offering content aligned with your brand, and then connecting the libraries in BrandIn. Of course, make sure you maintain assets on a regular basis to keep them aligned with your brand. My advice is to name a person on your team responsible for this task.

There are many pain points in presentation files. Users have been comp laining for years that it is difficult to check fonts in a file and replace them. Same with trying to figure out which layouts are used in which slides – it can take forever if you have a presentation that contains lots of slides. Using Neuxpower’s compression tools

I remember when the biggest struggle I had with presentations, or any o ther Microsoft Office file for that matter, was the huge file size. The first time I used Neuxpower’s compression tool, probably 18 years ago, I was blown away by how efficient it was at reducing the file size while making sure images remained high enough quality that the audience could not see the difference.

Since then, the product has evolved into five distinct products, which you ca n see on the products page on their website ( https://neuxpower.com/products ). In this section, I will introduce you to WeCompress (free) and NXPowerLite (paid). Even though this chapter is about PowerPoint add-ins, I feel any user who creates content should know about their compression tools.

If you don’t have great needs in terms of file compression and your files have a maximum size of 50 MB, you can then head to the wecompress.com website, or click on the product name on the Neuxpower p roducts page ( Figure 11.15 ):

Figure 11.15 – The WeCompress online file compression tool from Neuxpower

As you can see, you can compress the most common types of files used in organizations these days: PDF , PowerPoint , Word , Excel , JPEG , PNG , and TIFF files ( 1 ). You only need to click on the + sign ( 2 ) to browse your files and add one, or you can click and drag a file directly into the browser page. When the online application has finished processing your file, you will receive a File ready message ( 3 ). In our example, the initial 6.26 MB file size was brought down to 1.56 MB ( 4 ). The only thing left to do is click on the Download button ( 5 ) to move your compressed file into the Downloads folder on your computer.

The file I tested in the previous example showed the same photo quality as in my original file on all the slides. The only elements that seemed to be affected by the process were the small images created by the Zoom feature for my interactive menus. As you can see, it is very easy to use and has no cost at all.

If you have more frequent needs or your files are over 50 MB most of the time, I suggest that you invest in the NXPowerLite de sktop version ( Figure 11.16 ):

Figure 11.16 – Custom settings available in the NXPowerLite desktop compression application

You can create and name various optimization profiles ( 1 ) that will allow much more control over specific compression settings, such as the JPEG quality level ( 2 ) you want in your presentation files, or the creation of various custom settings for PDF , PowerPoint , Word , Excel , and JPEG files ( 3 ). A new setting allows removing Excel data from charts when using a strong compression profile ( 4 ).

NXPowerLite also integrates with File Explorer on Windows, with Finder on Mac, with Outlook, and much more. The license is a one-time, affordable purchase per user, and there is a 14-day trial available on their website. The desktop version installs an NXPowerLite group in the Home tab of your Office applications to help you optimize your files while you are creating them.

Even if the file compression tools are not PowerPoint add-ins per se, I’m sure you will find them useful when dealing with large presentations.

Build-a-Graphic is an add-in with a yearly subscription model, and you have the possibility to use a trial version with limited functionalities. This section will focus on the add-in with a subscription, but you can find all the information at the Build-a-Graphic website ( https://billiondollargraphics.com/build-a-graphic/ ).

You can use the Build-a-Graphic add-in in two ways:

  • Starting with your own bullet lists or text slides, build your own visuals with the help of the thousands of creative assets provided by the add-in
  • Search the library for prebuilt graphics tagged with keywords and add your own text content into the text boxes

Whether you choose to build your own or use a prebuilt graphic, the great advantage is that everything is fully customizable. Almost everything you add is a vector graphic, making it 100% editable.

You can change the colors, remove the fill colors, keep outlines, and ungroup shapes. Anything you can do in the Shape Format tab with PowerPoint’s native shapes can be used for Build-a-Graphic assets. If you feel you lack the skills to build your own graphics, and especially if you want to work wisely by not putting so much time into creating your own graphics, you should consider investing in this add-in.

It is worth mentioning that the Build-a-Graphic add-in was rebuilt from the ground up in collaboration with the Neuxpower people. This amazing version is -

  • Compatible with Windows, Mac, and web versions of PowerPoint.
  • Always connected to the complete library of graphics.

Creating your graphic from a text slide

The following example was created by Copilot, asking it to create a new slide for a four-step training process for success. If you don’t have a text slide on hand, just create a new blank slide using the title and content layout and add a list of four sentences.

When you want to test how text will look on a layout, you can use a nice hidden feature in your Office applications to create filler text. To create the Latin-looking text, you need to type =lorem(x,y) , where x is the number of paragraphs/bullets and y is the number of sentences per line, and press Enter .

If you want to reproduce readable text, you can use =rand(x,y) , which will generate the sentence The quick brown fox jumps over the lazy dog repeated for all paragraphs/bullets and sentences. Why this sentence? Because it uses all the letters in the alphabet, allowing you to see how your font displays. Be aware that in PowerPoint, the random text might be the same even if you are using another user interface language – that’s what I have in French. But in the French Word, the random text generated discusses Word features.

Using THOR (free)

The THOR add-in has one simple focus: memorize the size and position of an object on a slide so you can apply it – or hammer it, as Steve would say – to one or multiple selected objects elsewhere in your presentation. As for the name, yes, he is referring to the hammer-wielding god in Norse mythology, reflecting his great sense of humor and how cultured he is.

To install the add-in, simply navigate to the Free PPTools Add-ins section of the site (see Further reading ) and select THOR - The Hammer from the list. Then, follow the instructions in the Download and install THOR section. Once installed, you will see a new PPTools tab ( 1 ) in the ribbon (see Figure 11.25 ):

Figure 11.25 – The PPTools tab with the Hammer tool

The Thor group has a simple set of buttons. Here is a brief description of each:

  • Memorize ( 2 ): This is the button to click on first when you want to memorize the size and position of a slide object
  • Hammer ( 3 ): This is the button to click after selecting one or many other objects to which you want to apply the same size and position
  • Help ( 4 ): This button opens a .pdf file, providing all the information you need about the add-in without having to go back to the website
  • About ( 5 ): This provides access to regular licensing information, a link to the support, and a d iagnostics feature
  • Options ( 6 ): This b utton allows you to configure how THOR will work by changing the options in the THOR Options dialog box:
    • Alignment ( 7 ): Click on the radio button to choose which reference point to use. In this example, THOR will remember the position of the upper-left corner of an object.
    • Resize ( 8 ): This option allows you to remember the position of an object – if the checkbox is unchecked – or remember how to resize objects, as well as how to move them into the same position.
    • Don’t distort ( 9 ): This option is important when you decide to resize slide objects to avoid distortion.
    • Quiet mode ( 10 ): This option simply avoids bringing up a dialog box giving you the exact position and size of the memorized object. If you want to know more about this, please refer to the Help document.
    • And, of course, an OK button to confirm your settings.

Let’s see how THOR works with the following example in which we want to memorize the size and position of the circle on slide 2 ( 1 ) and apply it to the rectangle on slide 3 ( 2 ) ( Figure 11.26 ):

The final result is a resized rectangle ( 1 ) moved to the same position as the circle in slide 2 ( 2 ). The Don’t distort setting was on, so we s till have a rectangular shape ( Figure 11.27 ):

Using PPTMerge (paid)

The PPTMerge add-in is a great tool to help you merge data from an Excel file to create customized individual slides or complete presentations. If you are wondering how that can be helpful, here are a few examples:

  • Using a certificate PowerPoint template. You can then merge the list of names to show on each certificate.
  • Creating your social media posts by merging some key elements into a visual template created in PowerPoint. The credit for this amazingly creative idea goes to Catharine Richardson from My TechGenie (see Further reading ).

This time- saving add-in is available on the PPTools website in the products list as a one-time purchase. One testimonial on Steve’s PPTMerge page is from a client saying they only use it once a year, and it is still well worth the cost.
