Part 3

Execution: Strategy, Use Cases, and The Future

In this final part, you will bridge the gap between architectural theory and real-world application. You will begin by learning how to operationalize the patterns you’ve mastered, using a practical maturity roadmap to align technical implementation with organizational readiness. You will then step into a comprehensive, hands-on case study: building a loan processing system. You will follow the evolution of this system from a monolithic single-agent design to a robust multi-agent system, and, finally, implement these solutions using leading industry frameworks like Google ADK, CrewAI, and LangGraph. By the end of this part, you will have seen how abstract patterns translate into working code and will be equipped with a forward-looking perspective on the future of the agentic workforce.

This part of the book includes the following chapters:

  • Chapter 12, A Practical Roadmap: Implementing Agentic Patterns by Maturity Level
  • Chapter 13, Use Case: A Single Agent for Loan Processing
  • Chapter 14, Use Case: A Multi-Agent System for Loan Processing
  • Chapter 15, Agent Frameworks – Use Case: A Multi-Agent System for Loan Processing with CrewAI and LangGraph
  • Chapter 16, Conclusion: Charting Your Agentic AI Journey
